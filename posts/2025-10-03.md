# 安装

软件使用的是[Calibre-web](https://github.com/janeczku/calibre-web)，本来想用[reader3](https://github.com/hectorqin/reader)的，但是服务器太弱鸡不是很合适（单用户都要占用256m内存，多用户更是需要1G内存），所以只能退而求其次，毕竟只有看书的需求。

不过Calibre-web竟然有两个截然不同的分支，一个是[technosoft2000](https://github.com/Technosoft2000/docker-calibre-web)的，自带电子书转换，但是占用较大，另一个是[linuxserver](https://github.com/linuxserver/docker-calibre-web)的，相对更轻量，但是没有电子书转换功能。我这里选择了linuxserver的，想装technosoft2000的麻烦上网搜。

安装直接在终端git clone项目即可，终端跑完之后，去到/root/docker-calibre-web中，可以看到我们刚刚git clone的内容，新建一个docker-compose.yml文件，然后填入如下内容：

```bash
---
version: "2.1"
services:
  calibre-web:
    image: lscr.io/linuxserver/calibre-web:latest
    container_name: calibre-web
    environment:
      - PUID=0
      - PGID=0
      - TZ=Asia/Shanghai
      - DOCKER_MODS=linuxserver/mods:universal-calibre #optional
      - OAUTHLIB_RELAX_TOKEN_SCOPE=1 #optional
    volumes:
      - ./config:/config
      - ./books:/books
      - ./imports:/imports
    ports:
      - 8083:8083
restart: unless-stopped
```
填完之后，点开终端（在当前目录下），输入如下命令：

```bash
sudo docker-compose up -d
```

部署完毕之后，输入ip地址:8083即可成功访问：

![](https://r2.guange.top/%E6%90%AD%E5%BB%BA%E8%87%AA%E7%94%A8%E9%98%85%E8%AF%BB%E7%BD%91%E7%AB%99_01.webp)

然后在Location of Calibre Database这一栏中输入/books即可，如果弹出db错误，需要自己在PC端的Cailbre中复制metadata.db文件到/books目录下，或者到[这个地址](https://github.com/OzzieIsaacs/calibre-web-test/blob/main/test/Calibre_db/metadata.db)下载一个别人的metadb即可，成功后，右上角管理权限-管理用户可以调语言，最好在编辑Calibre数据库配置中把上传打开。

# 杂话

目前不开放注册，因为服务器比较菜鸡，防止恶意刷号恶意上传挤占资源，只开放浏览，我看过或者正在看的书籍都会放在这里，等我部署本地服务器（已经在计划中了）之后就会开放注册和上传供各位免费使用了。
