# 视频嵌入

这个主题本身不带嵌入插件，需要安装一个：

```bash
pnpm i @vrcd-community/markdown-it-video
```

安装完之后，在config文件头添加一行：

```bash
import markdownItVideo from "@vrcd-community/markdown-it-video";
```

再在MarkDown界面添加：

```bash
md.use(markdownItVideo, {
});
```

::: tip
需要有缩进，尽量与其他md模块保持一个缩进度
:::

之后在文章里输入这行命令就能实现嵌入视频了：

```bash
@[视频网站](视频ID)
```

::: tip
目前已经实现对 YouTube、Vimeo、Vine、Prezi、OSF、BiliBili等平台的适配
:::

示例：

@[bilibili](BV1uU421o7WJ)
